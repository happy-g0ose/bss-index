# BSS Index 🪿🍯

**BSS Index** — это интерактивный прайс-лист и калькулятор сделок для игры Roblox **Bee Swarm Simulator (BSS)** на русском языке.

Сайт помогает трейдерам оценивать стоимость стикеров, ваучеров и скинов, а также рассчитывать прибыльность обменов в реальном времени.

## Особенности 🌟
* 📊 **Калькулятор сделок**: добавьте предметы с двух сторон обмена и получите автоматический вердикт (*Слив*, *Убыточный трейд*, *Честный трейд*, *Выгодный трейд*, *Огромный плюс*).
* 📦 **Полная база предметов**: 289 игровых предметов с реальными изображениями, описаниями и рейтингами спроса.
* 📈 **Интерактивные графики**: визуализация изменения ценности предметов за последние 6 недель с помощью сглаженных SVG sparkline-графиков.
* ⚡ **Spotlight-поиск**: быстрый поиск предметов по нажатию `Ctrl + K` / `⌘K`.
* 🎨 **Премиальный дизайн**: Bento-стиль сетки с эффектом мышиного неонового свечения (Mouse Glow) под цвет редкости предмета.

## Технологический стек 🛠️
* React + TypeScript
* Tailwind CSS
* Framer Motion (анимации)
* Lucide React (иконки)
* GitHub Actions (автоматический деплой)
